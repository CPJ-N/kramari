# Kramari

**Open-source voice agent orchestration platform. Deploy production-ready AI voice agents in seconds.**

---

## What is Kramari?

A free, open-source alternative to Vapi that lets you build and deploy voice AI agents instantly. No vendor lock-in, run anywhere, customize everything.

## Why Kramari?

- **Deploy in seconds** - Voice agents up and running in under 10 seconds
- **Own your data** - Self-hosted, you control everything
- **Production-ready** - Handle real phone calls with AI-powered agents
- **Fully customizable** - Swap any component, extend however you want
- **100% open source** - MIT licensed, free forever

## What can you build?

- AI phone receptionists
- Customer service automation
- Sales qualification bots
- Support call handlers
- Voice-first applications

## Tech Stack

```
Frontend:  Next.js
Backend:   NestJS
Voice:     LiveKit
AI:        Claude (Anthropic)
```

## Quick Start

```bash
# Clone the repository
git clone https://github.com/kramari/kramari
cd kramari

# Run with Docker
docker-compose up

# Your voice agent platform is now running at localhost:3000
```

## Features

- **Real-time voice conversations** powered by LiveKit
- **AI orchestration** with Claude for natural interactions
- **Phone integration** for handling inbound/outbound calls
- **Agent builder** visual interface for creating voice workflows
- **API-first** integrate with your existing systems
- **Self-hosted** deploy on your own infrastructure
